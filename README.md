# DNS_Changer
DNS Changer

A simple and lightweight DNS management tool built with PowerShell.

DNS Changer makes changing your DNS settings on Windows faster and easier. No more going through multiple network settings menus — simply run the program as Administrator, choose your desired DNS configuration, and switch your DNS in seconds.

Features

- ⚡ Fast DNS switching
- 🔄 Restore network settings back to DHCP automatically
- 🖥️ Simple interactive command-line interface
- 🛠️ Built with PowerShell
- 🚀 No installation required

Whether you want to quickly change DNS servers for performance, troubleshooting, or network testing, DNS Changer provides a simple and efficient solution.
